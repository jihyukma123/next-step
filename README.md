# NextStep 프로젝트

README old에 비해서, 해결하고자 하는 문제에 더 집중하고자 구현 범위를 축소함. (Log/Review 카드 제거, 공부 주제 고민으로 인한 시간/에너지 낭비 문제 해결에 집중)

## 해결하려는 핵심 문제

### Problem A. 퇴근 후 "뭘 해야 할지" 결정이 안 나서 멍해짐

- '뭔가를 공부해야된다'라는 생각은 있음
- 근데 공부해야된다고 생각하는 주제가 너무 많음
- 그래서 책상에 딱 앉았을 때, 뭘 해야될지 몰라서 시간이 낭비됨

### Problem B. 공부를 해도 "기억이 안 남고 남는 게 없음"

- 이런저런 분야의 내용들을 잠깐씩 공부하고 기록하다보니 머릿속에 실력으로 쌓인다는 느낌이 없음

### Problem C. "성장하고 있다"는 감각이 없음

- 주제별로 진행 과정이 구조화되어 있지 않음
- 오늘 한 일이 목표와 연결되어 보이지 않음

---

## 사용 시나리오

1. **사이트 입장** → 오늘 추천 받기
2. **추천 확인** → 1~3개의 구체적인 학습 주제
3. **공부 실행** (앱 밖에서 진행)
4. **완료 기록** → 체크 + 한 줄 메모 (3초 컷)
5. **다음 방문** → 기록 기반으로 새로운 추천

**핵심:** 책상에 앉아서 "뭘 할까?" 고민하는 시간을 0으로 만든다.

---

## MVP 기능 (3개)

### 1. Topic 관리

- Topic 이름, 목표 1줄, 현재 상태 1줄
- 우선순위 (1~3)

### 2. 오늘의 추천

- 1~3개 추천 (룰 기반 + AI)
- 추천 기준: 최근 3일 안 한 것 우선
- 형식: "작고 구체적인 next action" (10~30분 단위)

### 3. 완료 기록

- ☑️ 체크
- 메모 (선택, 한 줄)
- 다음 추천에 자동 반영
